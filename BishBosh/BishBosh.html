Instructions

In order to play Bish Bosh, you need to enter 3 numbers.

the first number is the length of the loop that you would like to run. i.e 50.

The second number will replace every number dividable for the length of the loop by that number, by bish.

The third number will do the same as the second, expect this time if the number in the loop will be replaced by bosh.

if the number can be evenly divided by both bish and bosh, it will be replaced by bishbosh.

Example:
if first number is 15, second number or "bish" is 2 and third number or "bosh" is 3.

The result will be: 1, bish, bosh, bish, 5, bishbosh, 7, bish, bosh, bish, 11, bishbosh, 13, bish, bosh
